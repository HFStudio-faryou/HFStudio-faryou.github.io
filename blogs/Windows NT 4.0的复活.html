Windows NT 4.0的复活

Windows NT 4.0大家或许有些陌生，在95、98横行的年代，NT4.0在幕后默默当起了服务器的角色，不怎么引人注意，最后渐渐消失在了历史长河中。不过，NT4.0的NT内核却没有被微软放弃，之后的2000、XP、Vista等系统都是NT4.0的后代。今天，我将带领大家在2023年，让Windows NT 4.0展现它当年的风范！
