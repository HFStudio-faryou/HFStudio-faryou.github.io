Day3 安装补丁
先装SP1和SP2。
链接：https://www.123pan.com/s/6zDuVv-9nCHA.html
再装上贴吧大佬星游12制作的Windows Vista补丁合集（因为Windows Server 2008和Windows Vista同为Windows NT 6.0内核，所以Windows Vista能用的补丁Windows Server 2008基本也能用），按照压缩包内的说明安装即可。注意分清楚32位和64位。
链接：https://www.123pan.com/s/6zDuVv-QnCHA.html
然后打开Windows Update，接收剩余更新即可。
